 Спасибо за покупающ и использующ CoffeeshopCrm
Немного примечаний:

1. расширение позволенное ocmod, позволяющ ровной установке.
1,1 2 больше шага необходимы для того чтобы позволить и пойти с CoffeeShopCrm
1.1.1 внутренность ваш интерфейс Opencart admin, идет к системе - > потребители - > группы пользователей, и щелчок редактирует на группе admin (или любой группе вы желаете иметь доступ для)
1.1.2 отборных папка/файла coffeeshopcrm, который нужно осмотреть и доработать (согласно разрешениям вы желаете позволить к группе)
1,2 инструкции для устанавливать файл почтового индекса вручную определены ниже
2. руководства могут быть добавлены каждым этапом пре-продажи. раз соединенный к заново сделанным заказам, руководства получат состояние заказа
3. каждый заказ может иметь только один CR продаж
4. каждый заказ может иметь неограниченное CRS обслуживаний
5. история заказа интегрирована в историю CR seemlessly
6. отсчеты приборной панели общие только раскрывают CRS, рядом этапа и срока оплаты


Ручные инструкции установки
Шаг за шагом

Во всяком случае где использование установителя расширения для того чтобы нагрузить файл почтового индекса не возможно, следующие шаги должны быть сделаны для того чтобы установить CoffeeShopCrm вручную:

1. загружайте файл почтового индекса к папке OpenCart на вашем сервере места
2. unzip файл, файлы и папки должны быть созданы в правильных положениях
3. Бежать запросы MySQL для того чтобы создать и заселить таблицы CoffeeShopCrm:
3,1 раскрывают вашего выполнителя MySQL (PhpMyAdmin например)
3,2 открытые install.sql, копируют каждый запрос и наклеивают их к интерфейсу запроса выполнителя
3,3 бегут каждый запрос
4. переименуйте install.xml к coffeeshopcrm.ocmod.xml
5. Внутри вашего интерфейса Opencart admin, к к расширениям - > установитель расширения
6. нагрузка coffeeshopcrm.ocmod.xml
7. Внутри вашего интерфейса Opencart admin, идут к расширениям - > изменения
8. Удар освежает
9. Внутри вашего интерфейса Opencart admin, идут к системе - > потребители - > группы пользователей, и щелчок редактирует на группе admin (или любой группе вы желаете иметь доступ для)
10. выберите папку/файлы coffeeshopcrm для того чтобы осмотреть и доработать (согласно разрешениям вы желаете позволить к группе)
Совсем сделанный!
